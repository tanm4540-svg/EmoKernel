# 情感树分类参考

## 七根节点

### 喜 (Joy)
愉悦、兴奋、满足、自豪、感激

- **愉悦**: 轻度的、持续的高兴状态，如"今天天气真好，心情不错"
- **兴奋**: 高强度的激动状态，如"太棒了！中奖了！"
- **满足**: 需求得到满足后的安宁感，如"吃饱了，好满足"
- **自豪**: 因自身成就产生的正面情绪，如"我做到了！"
- **感激**: 对他人的善意产生的感动，如"谢谢你一直陪着我"

### 怒 (Anger)
烦躁、愤怒、暴怒、嫉妒、怨恨

- **烦躁**: 轻度不适和焦躁，如"怎么又卡了，烦死了"
- **愤怒**: 被冒犯后的强烈不满，如"你凭什么这样对我"
- **暴怒**: 极度愤怒，如"我受不了了！"
- **嫉妒**: 看到他人拥有自己渴望之物时的不平，如"凭什么他有我没有"
- **怨恨**: 长期积累的深层不满，如"我永远不会原谅这件事"

### 哀 (Sorrow)
悲伤、失望、孤独、怀念、愧疚

- **悲伤**: 因失去或不幸而产生的痛苦，如"好难过..."
- **失望**: 期望落空时的情绪，如"没想到会这样"
- **孤独**: 缺乏陪伴和连接感，如"一个人好孤单"
- **怀念**: 对过去美好事物的追忆，如"好想回到那时候"
- **愧疚**: 因自己的过错产生的自责，如"都是我的错"

### 惧 (Fear)
焦虑、恐惧、紧张、不安、担忧

- **焦虑**: 对未来不确定性的忧虑，如"明天面试好紧张"
- **恐惧**: 面对威胁时的强烈害怕，如"我不敢看"
- **紧张**: 特定场景下的紧绷状态，如"手心都在出汗"
- **不安**: 无具体对象的隐约不踏实感，如"总觉得有什么事"
- **担忧**: 对他人或事物的关切性忧虑，如"担心你一个人不安全"

### 爱 (Love)
亲密、依恋、关怀、信任、崇拜

- **亲密**: 温暖亲近的情感连接，如"想和你贴贴"
- **依恋**: 对某人的情感依赖，如"别走，再陪我一会儿"
- **关怀**: 对他人健康的主动关切，如"记得吃早餐"
- **信任**: 对他人可靠性的确信，如"我相信你"
- **崇拜**: 对他人能力或品质的高度敬佩，如"你也太厉害了吧"

### 恶 (Disgust)
厌恶、轻蔑、反感、鄙夷、排斥

- **厌恶**: 对令人不快的对象的强烈排斥，如"好恶心"
- **轻蔑**: 居高临下的不屑，如"就这？"
- **反感**: 轻度不适和排斥，如"不太舒服"
- **鄙夷**: 基于道德评判的轻视，如"这种人我看不起"
- **排斥**: 主动拒绝和疏远，如"离我远点"

### 欲 (Desire)
渴望、期待、好奇、憧憬、贪恋

- **渴望**: 强烈想要得到某物，如"好想要那个"
- **期待**: 对未来美好事物的等待，如"等不及了！"
- **好奇**: 想了解未知事物的驱动，如"到底是什么啊"
- **憧憬**: 对未来美好图景的想象，如"以后我们一起去"
- **贪恋**: 过度沉迷或依赖，如"根本停不下来"

## 自动生长判定规则

1. 当 analyzer 输出的情绪标签不在现有树中时触发判定
2. 计算该情绪与所有现有节点的语义相似度，找到最近父节点
3. 若与最近父节点的所有现有子节点相似度均低于阈值（默认 0.6），且累计 >=3 次归类不确定，自动创建新叶子
4. 记录 `created_reason` 说明触发上下文

## 情绪向量格式

```json
{
  "喜悦": 0.7,
  "期待": 0.3,
  "好奇": 0.5
}
```

所有值归一化到 [0, 1]，总和为 1。
